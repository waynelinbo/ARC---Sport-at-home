AWS IoT device SDK
##################

AWS IoT device SDK for embedded C is a collection of C source files which can
be used in embedded applications to securely connect to the AWS IoT platform.
It includes transport clients MQTT, TLS implementations and examples for their
use. It also supports AWS IoT specific features such as Thing Shadow. For more
information on the AWS IoT platform please visit the AWS IoT developer guide.
